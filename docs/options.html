References: Standard Keyword Options¶

PyPi: https://pypi.org/project/ffmpegio/
GitHub: https://github.com/tikuma-lsuhsc/python-ffmpegio

All read/write ffmpegio functions support a set of default keyword options.

Resolution start vs. end vs. duration¶

Only 2 out of 3 are honored.

start

end

duration

FFmpeg config

X

start time specified, continue till the end of input

X

start from the beginning of the input till end time is hit

X

start from the beginning of the input till encoded duration long

X

X

start and end time specified

X

X

start and duration specified

X

X

end and duration specified (actually sets start time)

X

X

X

end parameter ignored
